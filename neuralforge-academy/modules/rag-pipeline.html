🏭 The RAG Pipeline Factory

A RAG system has two main pipelines: Ingestion (offline) and Query (online). Documents flow through ingestion to build a knowledge base. Questions flow through query to generate grounded answers.

💻 RAG from Scratch

import numpy as np from sentence_transformers import SentenceTransformer import faiss class SimpleRAG: """RAG from scratch — no LangChain, pure fundamentals.""" def __init__(self, embedding_model="all-MiniLM-L6-v2"): self.embedder = SentenceTransformer(embedding_model) self.dim = 384 self.index = faiss.IndexFlatIP(self.dim) self.chunks = [] def ingest(self, documents, chunk_size=256): """Chunk, embed, and index documents.""" all_chunks = [] for doc in documents: words = doc.split() for i in range(0, len(words), chunk_size // 2): chunk = " ".join(words[i : i + chunk_size]) if chunk: all_chunks.append(chunk) embeddings = self.embedder.encode(all_chunks, normalize_embeddings=True) self.index.add(embeddings.astype(np.float32)) self.chunks.extend(all_chunks) def retrieve(self, query, k=3): """Find top-k relevant chunks.""" query_emb = self.embedder.encode([query], normalize_embeddings=True) distances, indices = self.index.search(query_emb.astype(np.float32), k) return [(self.chunks[i], float(d)) for d, i in zip(distances[0], indices[0])] def generate(self, query, k=3): """Full RAG: retrieve + generate.""" retrieved = self.retrieve(query, k) context = "\n\n".join([c for c, _ in retrieved]) prompt = f"""Answer based ONLY on this context: {context} Question: {query} Answer:""" return prompt

📊 RAG Evaluation

Faithfulness

Is the answer supported by retrieved context? Measures hallucination.

Context Precision

Are retrieved docs actually relevant? Measures retrieval quality.

Context Recall

Were all relevant docs retrieved? Measures coverage.

Answer Relevance

Does the answer address the question? Measures usefulness.
