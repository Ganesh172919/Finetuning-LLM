# SOTA AI — Learning Path

## Recommended Learning Order

### Phase 1: Foundations (Week 1-2)
1. `01_math/linear_algebra.py` — Vectors, matrices, tensors
2. `01_math/probability.py` — Distributions, entropy
3. `01_math/optimization.py` — Gradient descent, Adam
4. `01_math/calculus.py` — Derivatives, backpropagation

### Phase 2: Transformers (Week 3-4)
1. `02_transformers/attention.py` — Attention mechanisms
2. `02_transformers/embeddings.py` — RoPE, ALiBi
3. `02_transformers/layers.py` — RMSNorm, SwiGLU
4. `02_transformers/model.py` — Complete transformer

### Phase 3: Language Models (Week 5-6)
1. `03_llm/gpt.py` — GPT architecture
2. `03_llm/moe.py` — Mixture of Experts
3. `03_llm/reasoning.py` — Chain of Thought
4. `03_llm/slm.py` — Small Language Models

### Phase 4: Multimodal (Week 7-8)
1. `07_multimodal/clip.py` — CLIP
2. `08_vlm/vlm.py` — Vision Language Models
3. `09_image_generation/diffusion.py` — Diffusion models
4. `09_image_generation/vae.py` — VAE

### Phase 5: Advanced (Week 9-10)
1. `13_agentic_ai/react.py` — AI Agents
2. `14_rag/pipeline.py` — RAG
3. `19_reinforcement_learning/rlhf.py` — RLHF
4. `22_transformer_variants/mamba.py` — Mamba

### Phase 6: Production (Week 11-12)
1. `training/distributed.py` — Distributed training
2. `inference/engine.py` — Inference optimization
3. `deployment/serving.py` — Model serving

---

## Tips for Learning

1. **Read the docstrings** — Every file teaches concepts
2. **Run the code** — Each module has demonstrations
3. **Take notes** — Write down key insights
4. **Build projects** — Apply what you learn
5. **Read papers** — Follow the references
