The Problem: Traditional KV Cache

In standard attention, each request pre-allocates a contiguous block of memory for its maximum possible KV cache. This leads to massive waste.

The Solution: Paged Attention

Inspired by virtual memory in operating systems, Paged Attention divides KV cache into fixed-size blocks. Sequences can use non-contiguous physical blocks, eliminating internal and external fragmentation.

How Paged Attention Works

Block Table

Maps logical blocks to physical blocks. Each sequence has its own block table that tracks which physical memory blocks contain its KV cache.

# Block table mapping block_table = { "seq_A": [0, 3, 7], # Non-contiguous! "seq_B": [1, 4], # Shorter sequence "seq_C": [2, 5, 6, 8], # Longer sequence }

Copy-on-Write

When beam search needs to fork a sequence, Paged Attention uses copy-on-write: both beams share the same physical blocks until one needs to modify them. This saves significant memory.

# Beam search fork beam_1 = sequence.blocks # Shared reference beam_2 = sequence.blocks # Same blocks! # Only copy when writing if beam_2.needs_write: new_block = allocate_block() copy(block, new_block) # Now separate

# PagedAttention: Virtual Memory for KV Cache # Reference: vLLM (Kwon et al., 2023) class PagedKVCache: def __init__(self, num_blocks, block_size): # Physical memory: fixed-size blocks self.blocks = allocate_gpu_memory(num_blocks, block_size) self.free_blocks = set(range(num_blocks)) def allocate(self, seq_id): # Allocate one block from free pool block_id = self.free_blocks.pop() self.block_table[seq_id].append(block_id) return block_id def append_token(self, seq_id, k, v): # Get last block, allocate new if full last_block = self.block_table[seq_id][-1] if last_block.is_full(): last_block = self.allocate(seq_id) # Write KV to block (non-contiguous is fine!) last_block.append(k, v) def copy_on_write(self, src_seq, dst_seq): # Share blocks between sequences (beam search) self.block_table[dst_seq] = self.block_table[src_seq][:] # Mark as shared — copy only when writing for block_id in self.block_table[dst_seq]: self.ref_count[block_id] += 1
