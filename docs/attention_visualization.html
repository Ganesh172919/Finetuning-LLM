Multi-Head Attention (MHA)

Standard attention with H independent heads, each with its own Q, K, V projections.

Pros: Maximum expressiveness, each head learns different patterns

Cons: O(n²) memory for KV cache during inference

Formula: Attention(Q,K,V) = softmax(QK^T/√d)V

Grouped Query Attention (GQA)

Share KV heads across groups of Q heads. Balance between MHA (full) and MQA (single).

Pros: 2-4x faster inference than MHA, better quality than MQA

Cons: Slightly less expressive than MHA

Used by: LLaMA 2 70B, Mistral 7B

Flash Attention

IO-aware attention that tiles computation into GPU SRAM. Exact same result, 2-4x faster.

Key insight: Optimize for memory hierarchy, not FLOPs

Algorithm: Online softmax + tiled matmul

Memory: O(n) instead of O(n²)

Linear Attention

Replace softmax with kernel function to get O(n) complexity.

Approach: φ(Q)φ(K)^T instead of softmax(QK^T)

Trade-off: Linear complexity but may lose some quality

Used by: RWKV, Mamba (SSM-based)
