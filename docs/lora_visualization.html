LoRA — Low-Rank Adaptation

Add low-rank matrices A and B to frozen weights. Only A and B are trained.

Input x ─────────────────────────────────────────┬── Output
         │                                        │
         ├──▶ Frozen W (no gradient) ──▶ Wx ─────┤
         │                                        │
         └──▶ LoRA A ──▶ LoRA B ──▶ (α/r)·BAx ──┘
              (train)    (train)

Output = Wx + (α/r) · B · A · x

Why it works: Weight updates during fine-tuning are low-rank

Typical rank: 8-16 (most tasks), 64 (complex tasks)

QLoRA — Quantized LoRA

Combine 4-bit quantization with LoRA for extreme memory savings.

  • 4-bit NormalFloat: Quantize base model to 4-bit
  • Double Quantization: Quantize the quantization constants
  • Paged Optimizers: Handle memory spikes with CPU offload

Result: Fine-tune 65B model on single 48GB GPU!
