# SOTA AI — Frequently Asked Questions

### Q: What is this repository?
A: A comprehensive educational codebase implementing SOTA AI models from scratch.

### Q: Who is this for?
A: AI engineers, researchers, students learning modern AI.

### Q: Do I need a GPU?
A: No, all code runs on CPU. GPU optional for larger models.
