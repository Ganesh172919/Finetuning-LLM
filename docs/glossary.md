# SOTA AI — Glossary

## A

**Adam** — Adaptive Moment Estimation optimizer
**ALiBi** — Attention with Linear BiBiases
**ASR** — Automatic Speech Recognition
**Agent Teams** — Multiple AI agents collaborating on complex tasks (Claude 4.6)

## B

**BERT** — Bidirectional Encoder Representations from Transformers
**BPE** — Byte Pair Encoding (tokenization)

## C

**CLIP** — Contrastive Language-Image Pre-training
**CoT** — Chain of Thought prompting
**ControlNet** — Controllable image generation
**Claude 5** — Anthropic's latest model family (June 2026): Haiku 4.5, Sonnet 5, Opus 4.8, Fable 5, Mythos 5

## D

**DDPM** — Denoising Diffusion Probabilistic Models
**Deep Think** — Gemini 3's reasoning mode (math olympiad gold medal)
**DeepSeek R1** — Open-source reasoning model with MLA and GRPO training
**DPO** — Direct Preference Optimization

## F

**Fable 5** — Claude 5 Mythos-class model with safety guardrails
**Flash Attention** — Memory-efficient attention algorithm
**FSDP** — Fully Sharded Data Parallel

## G

**GELU** — Gaussian Error Linear Unit (activation)
**Gemini 3** — Google's latest model family with sparse MoE (128 experts)
**GPT** — Generative Pre-trained Transformer
**GPT-5** — OpenAI's model with auto-routing and native multimodal training
**GQA** — Grouped-Query Attention
**GRPO** — Group Relative Policy Optimization (DeepSeek R1 training)

## K

**KV Cache** — Key-Value Cache for efficient inference

## L

**LLM** — Large Language Model
**LoRA** — Low-Rank Adaptation

## M

**Mamba** — State Space Model architecture
**Mamba-2** — Structured State Space Duality (2-8x faster than Mamba-1)
**MLA** — Multi-head Latent Attention (DeepSeek's KV cache compression)
**MLM** — Masked Language Modeling
**MoE** — Mixture of Experts
**MoE-Mamba** — Mixture of Experts + Mamba (2.2x fewer training steps)
**MQA** — Multi-Query Attention
**Mythos 5** — Claude 5's most capable model (restricted access)

## N

**Nano Banana** — Google's viral image generation model (10M+ users)

## P

**PPO** — Proximal Policy Optimization

## R

**RAG** — Retrieval-Augmented Generation
**RLHF** — Reinforcement Learning from Human Feedback
**RMSNorm** — Root Mean Square Layer Normalization
**RoPE** — Rotary Position Embeddings

## S

**Safe Completions** — GPT-5's approach: safe responses instead of refusals
**SLM** — Small Language Model
**SSD** — Structured State Space Duality (Mamba-2's key insight)
**SSM** — State Space Model
**SwiGLU** — Swish-Gated Linear Unit

## T

**T5** — Text-to-Text Transfer Transformer
**TTS** — Text-to-Speech

## U

**U-Net** — Encoder-decoder architecture with skip connections

## V

**VAE** — Variational Autoencoder
**VLM** — Vision Language Model
