When to Use What

Use Transformer when:

  • Maximum quality is priority
  • Context length < 8K tokens
  • Hardware has ample memory
  • Using established tooling

Use RWKV when:

  • Long context is critical
  • Edge deployment needed
  • Streaming applications
  • Constant memory inference

Use Mamba when:

  • Very long sequences (100K+)
  • Hardware efficiency matters
  • Training on long documents
  • Genomics, time series

Use MoE when:

  • Scaling parameters is goal
  • Multi-task learning
  • Abundant GPU memory
  • Cost per token matters
