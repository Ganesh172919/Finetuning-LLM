# SOTA AI — Career Guide

## AI Career Paths

### Research Scientist
- Publish papers
- Advance the field
- PhD often required

### ML Engineer
- Build production systems
- Deploy models
- Strong engineering skills

### Data Scientist
- Analyze data
- Build models
- Business understanding

### AI Product Manager
- Define AI products
- Bridge tech and business

---

## Skills Needed

### Technical
- Python programming
- Deep learning frameworks
- Mathematics
- System design

### Soft Skills
- Communication
- Problem solving
- Teamwork

---

## Getting Started

1. **Learn fundamentals** — Math, programming
2. **Build projects** — Portfolio matters
3. **Contribute to open source** — Show your work
4. **Network** — Connect with others
5. **Stay current** — Field moves fast

---

*AI is a rapidly growing field with many opportunities.*
