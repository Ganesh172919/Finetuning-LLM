# SOTA AI — Interview Preparation

## Common Interview Questions by Topic

### Transformers
1. "What is self-attention?"
2. "Why do we need positional embeddings?"
3. "What's the difference between BERT and GPT?"
4. "How does Flash Attention work?"
5. "What is KV cache?"

### Large Language Models
1. "How do LLMs generate text?"
2. "What is temperature in sampling?"
3. "Explain top-k and top-p sampling"
4. "What is chain of thought prompting?"
5. "How does RLHF work?"

### Training
1. "What optimizer should I use for LLMs?"
2. "What is mixed precision training?"
3. "How do you do distributed training?"
4. "What is gradient accumulation?"
5. "How do you handle gradient explosion?"

### Image Generation
1. "How do diffusion models work?"
2. "What is latent diffusion?"
3. "What is the U-Net architecture?"
4. "How does ControlNet work?"
5. "What is LoRA?"

### Multimodal
1. "How does CLIP work?"
2. "What is cross-attention?"
3. "How do VLMs understand images?"
4. "What is contrastive learning?"

### Production
1. "How do you optimize LLM inference?"
2. "What is continuous batching?"
3. "How do you deploy models at scale?"
4. "What is speculative decoding?"

---

## Tips for AI Interviews

1. **Know the fundamentals** — Math, attention, transformers
2. **Understand tradeoffs** — Speed vs quality, memory vs compute
3. **Have examples ready** — Real projects, papers, implementations
4. **Stay current** — 2024-2026 developments
5. **Practice coding** — Implement key algorithms
